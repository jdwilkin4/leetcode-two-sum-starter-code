LeetCode Easy: Two Sum Visualizer

Description - Link to LeetCode problem

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target. You may assume that each input would have exactly one solution, and you may not use the same element twice. You can return the answer in any order.

Arr: [11, 15, 2, 7]

Target: 9

Output: [2,3]

Brute Force Approach

This approach involves checking each possible pair of numbers to find which pair adds up to the target and then returning the indices for that pair. Since this involves a nested for loop, then that would be O(n2) time complexity

Using a Map Approach

This approach involves a map and looping through the numbers array. For each iteration of the loop, we want to calculate the difference of subtracting the current number from the target.

If the difference is in the map, then that means we have found the pair of numbers we are looking for and can return the indices. Otherwise, we add the current number to the map along with its current index in the array.

Brute Force approach - O(n2) time complexity

Map solution - O(n) time complexity
